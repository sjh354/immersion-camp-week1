## 메뉴로 찾는 식당 검색 프로그램

원하는 메뉴가 있는 당신, 메뉴를 검색하라! 원하는 메뉴를 직접 입력해 음식점을 메뉴로부터 찾고, 가격이나 평점과 같은 정보를 얻을 수 있는 프로그램이다.

## 치킨을 원하면 치킨을 찾아라

- 메뉴를 먼저 검색해 원하는 메뉴를 찾아라!
- 스크롤을 하며 원하는 메뉴를 찾아라!
- 치킨, 면, 족발, 보쌈, 밥 등, ‘어디’가 어디서 먹을지를 결정해준다!

## 이거 좀 맛있어 보이는데?
